# IBM HR Analytics Dashboard

## Project Overview
Interactive Power BI dashboard analyzing employee attrition patterns and workforce demographics.

## Key Insights
- Overall Attrition Rate
- Attrition by Department
- Attrition by Job Role
- Overtime Impact on Attrition
- Gender Distribution
- Average Employee Age

## Tools Used
- Power BI
- DAX
- Data Visualization
- HR Analytics
